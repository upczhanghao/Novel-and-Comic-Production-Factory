## 1）林照野｜凡骨少年，筑基争命者

### 特征
- **背景**：山下小镇出身，父母早亡，靠替人跑腿、采药、修屋顶勉强长大。因灵根驳杂，被各大仙门视作“废苗”，却在宗门大选中被硬塞进争夺唯一筑基机缘的试炼队伍。
- **外貌**：十六七岁，身形偏瘦却筋骨利落，肤色常年日晒偏麦色，左眉尾有一道旧疤，眼睛黑亮，带一点不服输的野气。
- **性别**：男
- **年龄**：17
- **职业**：外门杂役 / 试炼候补 / 凡人少年
- **暗藏秘密或潜在弱点**：他体内有一缕极微弱却异常“干净”的古老灵息，会在濒死时自行护主；这灵息与失落的“仙魔旧约”有关。弱点是他对“被当成祭品”有极深恐惧，一旦察觉自己只是宗门棋子，会产生短暂失控与自毁倾向。

### 核心驱动力三角
- **表面追求**：抢到筑基机缘，活着离开试炼。
- **深层渴望**：证明自己不是可被随手碾碎的凡骨，想让曾经看不起自己的人承认他。
- **灵魂需求**：建立一种“人不必先天高贵才配被尊重”的信念，守住作为人的底线。

### 语言画像
- **说话风格**：短句多，常先停顿一下再说；被逼急时会冒出一句很直的反问。
- **用词倾向**：口语化、带一点乡下人的朴素比喻。
- **情绪表达方式**：平时克制，情绪上来时会突然变得锋利。
- **对不同对象的语言差异**：
  - **对上级**：表面恭敬，实则每句话都留一点余地，“弟子明白……只是，真要这么做吗？”
  - **对朋友**：直接，偶尔带点自嘲，“我命硬，死不了。”
  - **对敌人**：言简意赅，带挑衅，“你想拿我当垫脚石？先看看脚底够不够稳。”
  - **对爱人**：笨拙，真诚，话反而更少，“别怕，我在。”
- **独有语言标识**：喜欢把生死局说成“**赌命的活**”。

### 角色弧线设计
- **初始状态** → 只想活命、只信拳头和运气。
- **触发事件** → 被宗门长老点名参与唯一筑基争夺，并发现落败者会被秘密献祭。
- **认知失调** → 一边渴望筑基翻身，一边无法接受用他人性命换自己的前途。
- **蜕变节点** → 在试炼最深处目睹“凡人祭品”真相后，反手破坏宗门布置，选择先救人再争机缘。
- **最终状态** → 不再把筑基当成唯一答案，成为既能逆天改命、又敢为凡骨立命的修者。

### 关系冲突网
- **与其他角色的关系或对立点**：
  - 与宗门高层对立：他是“可消耗资源”与“人”的分界线。
  - 与同届天骄对立：对方视他为拖累，他视对方为“拿命换名”的怪物。
- **与至少两个其他角色的价值观冲突**：
  - **与苏绫霜**：她信秩序和大局，他信底线和个体。
  - **与谢无咎**：对方只信结果与代价，他拒绝把人当材料。
- **一个合作纽带**：和阿朔共享凡人出身的生存经验，能在试炼中形成默契。
- **一个隐藏的背叛可能性**：当他确认宗门要拿阿朔或自己献祭时，可能会违令单独行动，表面看像背叛队伍，实则是切断宗门棋局。

## 2）苏绫霜｜寒门剑修，秩序的守门人

### 特征
- **背景**：名门旁支出身，幼年受门规与家法双重约束，靠极高悟性与近乎苛刻的自律爬入内门。曾亲历宗门大乱，认定“无序即灾”。
- **外貌**：二十三四岁，身姿高挑，面容清冷，眉眼细长如刃，常着素白剑袍，发束得一丝不乱。
- **性别**：女
- **年龄**：24
- **职业**：内门剑修 / 试炼监察者
- **暗藏秘密或潜在弱点**：她的本命剑曾在仙魔旧约震荡中裂过一次，裂痕未愈，若情绪失控或遭旧约之力冲击，剑心会反噬。她秘密参与过一次“清理祭品”的行动，至今背负罪责。

### 核心驱动力三角
- **表面追求**：维持宗门秩序，确保筑基机缘落到“最合适”的人手里。
- **深层渴望**：得到一个无需时刻证明自己正确、无需再背负牺牲者目光的安稳世界。
- **灵魂需求**：承认“秩序不是替罪羊的借口”，学会在规则与良知之间选择后者。

### 语言画像
- **说话风格**：句式整齐，像下命令；很少说废话。
- **用词倾向**：文雅、克制、偏宗门典籍用语。
- **情绪表达方式**：压抑隐忍，越愤怒越平静。
- **对不同对象的语言差异**：
  - **对上级**：极其规矩，“弟子领命，必不误事。”
  - **对朋友**：会多一点人味，但仍然简短，“活着回来，别让我去收尸。”
  - **对敌人**：冷到像宣判，“你已越界，退无可退。”
  - **对爱人**：极少直白，更多是替对方安排后路，“若我不在，你可按此路线离开。”
- **独有语言标识**：常把结论落在一句“**此事当止于此**”。

### 角色弧线设计
- **初始状态** → 把宗门秩序视作最高准则。
- **触发事件** → 监察试炼时，发现筑基机缘实为筛选祭品的幌子。
- **认知失调** → 既想维持稳定，又无法忽视被牺牲者的真实痛苦。
- **蜕变节点** → 在林照野选择救人后，她亲手斩断一条“合规却残酷”的命令链。
- **最终状态** → 成为能为规则开裂口的人，不再是秩序的盲目执行者，而是秩序的修补者。

### 关系冲突网
- **与其他角色的关系或对立点**：
  - 初期视林照野为不稳定变量，后期成为最难割舍的盟友。
  - 与谢无咎关系紧张：一个要规则，一个要效率。
- **与至少两个其他角色的价值观冲突**：
  - **与林照野**：她重整体稳定，他重个体尊严。
  - **与谢无咎**：她承认代价但不接受彻底的工具化，他则不在乎手段。
- **一个合作纽带**：她掌握试炼阵图与宗门通行法，能为众人打开生路。
- **一个隐藏的背叛可能性**：若宗门以“保全苍生”为名逼她处决林照野，她可能假意执行，实则放他离开并伪造结果。

## 3）谢无咎｜药师出身的谋局者，结果至上

### 特征
- **背景**：出自落败的炼药世家，家族在灵脉枯竭后转而依附宗门，靠研究禁药与残方苟存。谢无咎自幼见惯病死与饥荒，因此对“过程是否体面”毫无执念。
- **外貌**：二十八九岁，面相清俊却常有疲态，眼下淡青，手指修长，袖口总沾着药灰与墨迹，穿灰黑长衫。
- **性别**：男
- **年龄**：29
- **职业**：丹师 / 试炼随行医官 / 密谋者
- **暗藏秘密或潜在弱点**：他研究过“活祭转灵”之术，知道筑基机缘背后需要献祭凡骨；他也曾用自家族人的命试药，内心深处其实极怕有人用同样的方式审判他。弱点是重度失眠，幻听严重，越接近旧约崩裂的灵脉异动越容易判断失准。

### 核心驱动力三角
- **表面追求**：夺取筑基资源样本，完善丹道与续命之法。
- **深层渴望**：找到一种不靠牺牲亲人和弱者也能活下去的方法。
- **灵魂需求**：面对自己“用代价换成果”的人生账本，重新定义什么叫真正的救人。

### 语言画像
- **说话风格**：条理分明，常像在下诊断；喜欢先抛结论再补原因。
- **用词倾向**：技术性很强，夹杂医理、丹理、阵理术语。
- **情绪表达方式**：表面冷静，内里焦躁时会变得刻薄。
- **对不同对象的语言差异**：
  - **对上级**：谨慎、数据化，“成功率三成，若要稳妥，需再添两味材料。”
  - **对朋友**：会用半玩笑掩饰关心，“别死太快，我还没把你值钱的地方榨干。”
  - **对敌人**：讥讽且精准，“你这种人，连入药都嫌药性冲。”
  - **对爱人**：少见温柔，更多是实用式关怀，“把手伸来，先止血。”
- **独有语言标识**：习惯把人的问题归结成“**脉象不对**”。

### 角色弧线设计
- **初始状态** → 只相信代价、效率与结果。
- **触发事件** → 在试炼中检出“祭品阵”而被迫救治重伤者，第一次近距离面对自己参与的罪。
- **认知失调** → 明知残酷之法有效，却开始怀疑“有效”是否等于“对”。
- **蜕变节点** → 亲手销毁能延续献祭体系的关键丹方，转而与林照野联手破局。
- **最终状态** → 从“拿人试药的人”变成“替人找活路的人”，仍然现实，却不再冷血。

### 关系冲突网
- **与其他角色的关系或对立点**：
  - 与林照野互相利用又互相拯救。
  - 与苏绫霜长期理念冲突，但在危局中能形成高效配合。
- **与至少两个其他角色的价值观冲突**：
  - **与林照野**：他接受代价逻辑，林照野拒绝。
  - **与阿朔**：阿朔对“活下去”有朴素伦理，他则常把伦理让位于实验。
- **一个合作纽带**：他掌握灵脉枯竭的药理知识，能救阿朔与其他凡人试炼者的命。
- **一个隐藏的背叛可能性**：若发现旧约裂缝中存在重建灵脉的唯一代价是献祭林照野体内那缕古老灵息，他可能先隐瞒真相，试图独自完成替代方案。

## 4）阿朔｜杂役同伴，最先看见真相的人

### 特征
- **背景**：山林猎户之子，小时候所在村子被灵脉暴动波及，整个村只剩他一人活下来，因此被宗门收作最低等杂役。虽无灵根，却对山林、毒草、风向、脚印极其敏感。
- **外貌**：十五六岁，个子不高，脸圆却不显稚气，眼神机灵，常穿旧布衣，袖口和裤脚总有草屑与泥印。
- **性别**：男
- **年龄**：16
- **职业**：杂役 / 试炼搬运工 / 地形向导
- **暗藏秘密或潜在弱点**：他能听见灵脉枯竭时的“低鸣”，类似某种天生异感；这使他能提前感知灾变，但也会因此头痛欲裂。弱点是极度害怕被抛下，一旦觉得自己没价值就会做出过度冒险的事。

### 核心驱动力三角
- **表面追求**：混进试炼队伍，活着拿到赏赐。
- **深层渴望**：有一次被真正当作“同伴”而不是工具。
- **灵魂需求**：相信自己活下来不是侥幸，也不是为了给别人当垫脚石。

### 语言画像
- **说话风格**：快、碎、常插话，爱把严肃话说得像闲聊。
- **用词倾向**：口语化、带山里孩子的直觉比喻。
- **情绪表达方式**：故作轻松，害怕时会嘴硬。
- **对不同对象的语言差异**：
  - **对上级**：缩着说话，尽量不惹麻烦，“我、我就搬东西，别让我上前线行不？”
  - **对朋友**：热络，能说个不停，“我认路，真认路，闭着眼都能摸回去！”
  - **对敌人**：一旦急了会非常狠，“你敢动我哥，我咬死你。”
  - **对爱人**：笨拙又坦白，“我没啥本事，但我会一直陪着你。”
- **独有语言标识**：把危险前兆叫做“**山风变味了**”。

### 角色弧线设计
- **初始状态** → 只求不被淘汰、不被遗忘。
- **触发事件** → 在试炼里意外发现祭坛机关，并听见灵脉低鸣预示大灾。
- **认知失调** → 他既想活命，又不想眼睁睁看同伴被献祭。
- **蜕变节点** → 主动带队绕开死路，第一次把“自己也能救人”当成事实。
- **最终状态** → 从被保护的杂役成长为敢于冲在最前面、能把众人从崩坏山脉里带出来的关键活口。

### 关系冲突网
- **与其他角色的关系或对立点**：
  - 与林照野是最早形成生死默契的人。
  - 被苏绫霜起初当作不够稳定的杂役，后期逐渐认可。
- **与至少两个其他角色的价值观冲突**：
  - **与谢无咎**：他讲情义与直觉，谢无咎讲代价与结果。
  - **与宗门天骄**：对方看不起凡人，他因此本能敌视。
- **一个合作纽带**：他能感知地势与灵脉异常，是整个小队的活地图。
- **一个隐藏的背叛可能性**：若宗门承诺放过他故乡残存的亲人，他可能被迫泄露队伍路线，但最终大概率会反悔。

## 5）闻人策｜宗门长老，旧约的维护者

### 特征
- **背景**：执掌宗门刑律与祭祀多年，亲眼见过灵脉由盛转枯，也见过仙魔旧约压制下的短暂和平。他坚信“大义必须有人承担血债”。
- **外貌**：五十余岁，须发半白，衣冠整肃，手执青铜拂尘，眉目温和却无端让人发冷。
- **性别**：男
- **年龄**：57
- **职业**：宗门长老 / 刑律执掌者 / 祭阵总管
- **暗藏秘密或潜在弱点**：他曾在上一次旧约动荡中亲手送走一
